\# MCP Server Setup



Connect AI tools to CDP documentation using the Model Context Protocol for enhanced development assistance



\## Overview

Think of MCP (Model Context Protocol) as giving your AI assistant a direct hotline to Coinbase's documentation...



Without MCP:

You: "Create a wallet connection component"

AI: generates generic code



With MCP:

You: "Create a wallet connection component"

AI: searches CDP docs and generates correct code



\## How it helps you

• Generate accurate code using CDP components  

• Answer questions about CDP APIs  

• Suggest best practices for embedded wallets  

• Find documentation when stuck
